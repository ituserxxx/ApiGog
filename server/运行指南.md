
## 功能验证

- 后端健康检查：http://localhost:5000/api/health
- 前端页面：http://localhost:3000
- API代理：前端通过 `/api` 路径代理到后端

## 技术栈

- 后端：Flask 3.0 + Flask-CORS
- 前端：Vue 3 + Vite + Axios
- 代理：Vite开发服务器代理API请求到Flask
